## Frontmatter Schema

All blog posts must include:

```yaml
---
title: "String - Required"
date: "YYYY-MM-DD" - Required
description: "String - Required (1-2 sentences)"
slug: "kebab-case-string - Required"
published: true - Boolean (default: true)
tags: ["Array", "Of", "Strings"] - Optional
author: "String - Optional (default: Dao Quang Truong)"
coverImage: "URL - Optional"
updatedDate: "YYYY-MM-DD" - Optional
---
```

## Content Guidelines

### Writing Style
- **Clear and concise**: Avoid unnecessary complexity
- **Practical**: Focus on actionable content
- **Examples**: Include code snippets where relevant
- **Scannable**: Use headers, lists, and tables
- **Professional yet approachable**: Technical but readable

### Code Examples
- Always specify language: \`\`\`typescript
- Keep examples focused and minimal
- Include comments for complex parts
- Show both good and bad examples when helpful

### Formatting
- **Headers**: `## Section`, `### Subsection`
- **Bold**: `**important**` for emphasis
- **Italic**: `*terms*` for definitions
- **Lists**: `-` for items, `1.` for ordered
- **Tables**: Use for comparisons
- **Callouts**: Use for warnings/tips

### Images
- Use Unsplash with proper parameters
- Minimum width: 1000px
- Format: `?auto=format&fit=crop&q=80&w=1000`

### Links
- Relative links for internal: `[text](/path)`
- Absolute with protocol for external: `[text](https://...)`
- Always include descriptive link text

## AI Agent Instructions

When generating blog posts:

1. **Choose the right template** based on content type:
   - Real project experience → Case Study
   - Technical deep dive → Advanced Patterns
   - Personal learning → Reflection
   - Step-by-step guide → Tutorial

2. **Always use English** for all content

3. **Follow frontmatter schema exactly**

4. **Maintain consistent formatting**:
   - Headers hierarchy
   - Code block syntax
   - List styles
   - Table format

5. **Tags**: Use 2-4 relevant tags from existing categories

6. **Slug**: Kebab-case, descriptive, matches file name

7. **Cover image**: Use Unsplash with appropriate topic

8. **Length**: 
   - Case Study: 800-1500 words
   - Advanced Patterns: 1500-2500 words
   - Reflection: 500-1000 words
   - Tutorial: 1000-2000 words
